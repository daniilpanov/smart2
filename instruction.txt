1. Создать базу данных для умного дома, импортировать sql файл из архива
2. На сервере, открыть папку Users/adminwsr/html
3. В этой папке, должно быть много файлов, если нет, то перезагружать сервер, пока не будет
4. Создать в этой папке папку smart-home или с любым другим названием
5. Поместить туда содержимое бекэнда из архива
6. Изменить подключение к базе данных (config/db.php)
7. На сервере или по SSH изменить группу владельца созданной папки: chown -R adminwsr:adminwsr smart-home
8. В базе данных умного дома 0 пользователей, чтобы их завести нужно вызвать консольную команду из корня бекэнда: ./yii hello/generate-users X, где X - кол-во участников, которые вам нужны
9. В таблице users задать логины и пароли участникам из сервера, для авторизации участников

Также в архиве есть другой архив (smart home test), который поможет протестировать работоспособность бекэнда, его можно положить в любую папку участника, изменить в файле main.js ip адрес сервера и логин/пароль одного из участников. Открыть эту страницу в браузере и в консоли должны быть отчеты о тестах (авторизован, список комнат, комната и т.п ..)
